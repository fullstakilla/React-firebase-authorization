## Быстрый старт

1.  **Склонируйте репозиторий.**

    ```sh
    git clone https://github.com/fullstakilla/React-firebase-authorization
    ```

2.  **Начните просмотр.**

    Перейдите в корневую директорию и запустите проект.

    ```sh
    yarn install
    yarn dev
    ```
